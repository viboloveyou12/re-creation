# 北科文發系105級畢業展覽官網

## 簡介
台北科技大學文化發展系105級學生畢業展覽形象官網。<br>
文化的世界裡充滿了各式各樣的驚喜，
就如同我們熱帶雨林中冒險，也許會看見聳立千年的大樹；也許會聽見此起彼落清脆悅耳的聲音；
我們也會發現裡頭的生機盎然，而文化的不同面向幻化成豐富的生命，連接著世代又不斷的創新。

「創甡」分為四大主軸，展覽於四個展館當中，訴說著各式各樣相當精彩的故事。
